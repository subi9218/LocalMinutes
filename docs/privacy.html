적자생존 개인정보 처리방침

시행일: 2026년 5월 1일

적자생존은 사용자의 회의 내용을 보호하기 위해 음성 인식, 발화자 라벨, 회의록 요약을 사용자의 Mac 안에서 처리하도록 설계된 macOS 앱입니다. 이 방침은 적자생존이 어떤 정보를 처리하고, 어디에 저장하며, 언제 네트워크를 사용하는지 설명합니다.

1. 핵심 원칙

  • 회의 음성, 전사 텍스트, 요약 내용은 기본적으로 사용자의 Mac 안에서만 처리됩니다.
  • 적자생존은 회의 음성, 전사본, 요약본을 개발자 서버로 업로드하지 않습니다.
  • 적자생존은 광고 추적, 사용자 프로파일링, 제3자 광고 SDK를 사용하지 않습니다.
  • 사용자가 직접 내보내기, 공유, 메일 발송, 로그 복사 등을 실행한 경우에는 사용자가 선택한 대상 앱 또는 서비스로 해당 내용이 전달될 수 있습니다.

적자생존은 앱 기능 제공을 위해 다음 정보를 사용자의 Mac에 저장하거나 처리할 수 있습니다.

3. 로컬 AI 처리

적자생존은 음성 인식과 요약을 클라우드 API가 아니라 사용자가 설치한 로컬 모델로 처리합니다.

  • 음성 인식 모델은 녹음 파일을 텍스트로 변환합니다.
  • 요약 모델은 전사 텍스트와 사용자가 입력한 메모를 바탕으로 회의 요약, 결정사항, 액션아이템 등을 생성합니다.
  • 발화자 라벨 모델은 사람 이름을 식별하지 않고, 발화 구간을 A/B/C 같은 라벨로 구분하는 데 사용됩니다.

앱스토어용 기본 빌드에서는 상업 배포에 라이선스 리스크가 있는 모델을 숨기며, 설정 화면에서 사용 모델 및 라이선스 정보를 확인할 수 있습니다.

4. 네트워크 사용

적자생존은 기본 회의 처리 과정에서 회의 내용을 개발자 서버로 전송하지 않습니다. 다만 다음 경우에는 네트워크가 사용될 수 있습니다.

  • 사용자가 앱 안에서 음성 인식, 요약, 발화자 라벨 모델을 다운로드할 때 Hugging Face 또는 GitHub 같은 모델 제공 사이트에 접속합니다.
  • 사용자가 모델 제공 사이트의 토큰을 입력하는 경우, 해당 토큰은 모델 다운로드 요청에만 사용됩니다.
  • 사용자가 도움말, 라이선스, 외부 링크를 여는 경우 기본 브라우저가 열릴 수 있습니다.
  • 사용자가 macOS 공유 시트, 메일 앱, Notion, Slack 등 외부 앱으로 회의록을 직접 공유하는 경우 해당 외부 앱 또는 서비스의 개인정보 처리방침이 적용됩니다.

적자생존은 다음 macOS 권한을 사용할 수 있습니다.

권한사용 이유

네트워크 접근사용자가 요청한 모델 파일 다운로드와 외부 링크 열기에 필요합니다.

앱스토어용 기본 빌드에서는 Calendar.app 자동화 및 AppleEvent 권한을 사용하지 않습니다.

6. 로그와 진단 정보

적자생존은 앱 오류 분석을 위해 로컬 진단 로그를 남길 수 있습니다. 이 로그는 사용자의 Mac 안에 저장되며 자동으로 개발자에게 전송되지 않습니다.

로그에는 오류 메시지, 앱 내부 처리 단계, 스택 트레이스 등이 포함될 수 있습니다. 오류가 회의 처리 중 발생한 경우 파일 경로, 회의 제목, 일부 오류 문맥이 포함될 수 있으므로, 사용자가 로그를 공유하기 전에 내용을 확인하는 것을 권장합니다.

사용자는 설정 화면에서 로그 내용을 확인, 복사, 삭제할 수 있습니다.

7. 사용자가 직접 공유하는 정보

사용자가 다음 기능을 실행하면 선택한 대상 앱 또는 서비스로 회의 정보가 전달될 수 있습니다.

  • 회의록 내보내기
  • PDF, Markdown 등 파일 저장
  • macOS 공유 시트 사용
  • 이메일 작성
  • 로그 복사 또는 공유
  • Finder에서 파일 열기

이 경우 적자생존은 사용자가 선택한 동작을 수행할 뿐이며, 전달된 정보는 대상 앱 또는 서비스의 정책을 따릅니다.

8. 데이터 보관과 삭제

  • 녹음 파일은 사용자가 선택한 저장 폴더에 저장됩니다.
  • 전사, 요약, 회의 메타데이터는 앱의 로컬 데이터베이스에 저장됩니다.
  • 사용자는 앱 안에서 회의록을 삭제하거나, 저장 폴더의 녹음 파일을 직접 삭제할 수 있습니다.
  • 자동 삭제 기능을 켠 경우, 설정한 기간보다 오래된 녹음 파일이 삭제될 수 있습니다. 회의록, 전사, 요약 데이터는 별도로 유지될 수 있습니다.
  • 앱을 삭제해도 사용자가 선택한 외부 저장 폴더에 있는 녹음 파일은 macOS 정책에 따라 남아 있을 수 있습니다.

9. 제3자 제공

적자생존은 개발자 서버를 통해 회의 데이터, 전사본, 요약본을 제3자에게 판매하거나 제공하지 않습니다.

사용자가 직접 외부 앱으로 공유하거나 모델 제공 사이트에서 모델을 다운로드하는 경우, 해당 외부 서비스와의 통신은 사용자의 선택에 따라 발생합니다.

10. 아동의 개인정보

적자생존은 어린이를 대상으로 설계된 앱이 아니며, 어린이의 개인정보를 의도적으로 수집하지 않습니다.

11. 보안

적자생존은 회의 데이터를 로컬 Mac에 저장하고, 외부 서버 전송을 기본 동작에 포함하지 않는 방식으로 개인정보 노출 위험을 줄입니다. 다만 사용자의 Mac 계정, 저장 폴더 접근 권한, 백업 서비스, 외부 공유 앱의 보안 설정은 사용자 환경에 따라 달라질 수 있습니다.

민감한 회의의 경우 다음을 권장합니다.

  • 신뢰할 수 있는 저장 폴더를 선택합니다.
  • macOS 계정 잠금과 디스크 암호화를 사용합니다.
  • 회의록 또는 로그를 외부 앱으로 공유하기 전에 내용을 확인합니다.
  • 더 이상 필요 없는 녹음 파일은 삭제합니다.

12. 방침 변경

개인정보 처리방침이 변경될 경우 앱 업데이트, 앱스토어 설명, 또는 배포 페이지를 통해 변경 내용을 안내할 수 있습니다.
